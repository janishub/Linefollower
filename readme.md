## specifications
<br />
microcontroller: esp32
<br />
motors: polulu 6v 300rpm 50:1
<br />
h-bridge: adafruit drv 8833
<br />
sensors: qt8a
<br />
batteries: li-ion 3,7VDC (2x)
<br />
wireless communication: bleuthooth
<br />
distance sensor - motors: 10 cm
<br />
weight: 248gram 
<br />
speed: 0.502 m/s
